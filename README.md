# vim-wrapper
Adds quotes and brackeds around a selected block of text

## Usage

Select a block of text then:

- `q'` - to wrap it with `'text'`
- `q"` - to wrap it with `"text"`
- `q(` or `q)` - to wrap it with `(text)`
- `q[` or `q]` - to wrap it with `[text]`
- `q{` or `q}` - to wrap it with `{text}`
